# SessionTrack· Focus Analytics App

Track your focus sessions, analyze productivity patterns, and compete with others — all in one dashboard.

## ✨ Features
- Focus timer with session categories  
- Analytics with weekly and monthly insights  
- Focus calendar with streak tracking  
- Leaderboard to compare productivity  
- Session history with filtering and search  
- Customizable user profiles  
- Dark mode support  
- Secure authentication  
- Cloud-based data storage  

## 🚀 Live Demo
https://gouravk1107.github.io/sessiontrack/

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript (Vanilla)  
- Backend: Firebase Authentication + Firestore  
- Hosting: GitHub Pages  

## 📁 Structure
- Dashboard, analytics, sessions, leaderboard, profile pages  
- Modular CSS and JavaScript files  
- Firebase config handled separately  

## 🔧 Setup
- Create a Firebase project  
- Enable Authentication and Firestore  
- Add your Firebase config locally  
- Run using a local server  

## 🚀 Deployment
- Push to GitHub  
- Enable GitHub Pages  
- Site goes live instantly  

## 📊 Data Model
- Users: profile, bio, stats  
- Sessions: time, category, task, duration  

## 🎯 Core Flow
- Sign up → start focus session → track time → view analytics → compete on leaderboard  

## 🔒 Security
- User-specific data access  
- Authentication required  
- Sensitive config excluded from repo
